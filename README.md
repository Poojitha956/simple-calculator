# Simple Calculator

A beginner-friendly calculator project built using Python.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Power (^)
* Percentage / Modulus (%)
* Square Root (sqrt)

## Technologies Used

* Python
* VS Code
* Git
* GitHub

## How to Run the Project

1. Open the project folder in VS Code
2. Open Terminal
3. Run the following command:

```bash
python calculator.py
```

## Example Usage

```text
Choose (+, -, *, /, ^, %, sqrt): +
Enter first number: 10
Enter second number: 5
Result is: 15.0
```

## Square Root Example

```text
Choose (+, -, *, /, ^, %, sqrt): sqrt
Enter number: 25
Square root: 5.0
```
